# Data Source Information

## Dataset
**Brazilian E-Commerce Public Dataset by Olist**

## Source
[Kaggle: Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)

## Description
Anonymized data on ~100,000 orders made at the Olist Store in Brazil from 2016 to 2018, covering customer behavior, sellers, products, payments, deliveries, and reviews across all Brazilian states.

## Files Included (9 CSV files)
- `olist_customers_dataset.csv` - Customer information and locations
- `olist_orders_dataset.csv` - Order metadata, status, and timestamps
- `olist_order_items_dataset.csv` - Items per order with prices and sellers
- `olist_order_payments_dataset.csv` - Payment information per order
- `olist_order_reviews_dataset.csv` - Customer reviews and ratings
- `olist_products_dataset.csv` - Product details and categories
- `olist_sellers_dataset.csv` - Seller information and locations
- `olist_geolocation_dataset.csv` - Brazilian zip code geocoordinates
- `product_category_name_translation.csv` - Portuguese to English category name mappings

## Files Used in This Analysis (6 of 9)

## Setup Instructions
1. Download the full dataset from the Kaggle link above
2. Place all CSV files in this `data/` folder
3. Run the dashboard: `streamlit run dashboard/app.py`
