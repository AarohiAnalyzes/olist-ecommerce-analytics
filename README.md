# Olist Brazilian E-commerce Analysis

## 📊 Project Overview
Comprehensive analysis of Brazilian e-commerce data covering sales, customer, and product insights.

## 🎯 Business Questions
1. **Sales Performance**: Monthly revenue trends and order patterns
2. **Customer Analysis**: Geographic distribution and segmentation
3. **Product Insights**: Category performance and seller efficiency

- `docs/` - Project documentation
- `notebooks/` - Analysis notebooks (coming soon)
